Este repositório contém os códigos de algoritmos de escalonamento não-preemptivos dos tipos: <br/>
- FIFO (First In First Out), no qual os processos são executados segundo a ordem que são chamados; e
- SJF (Shortest Job First), no qual o processo em estado de pronto que tiver menor tempo de execução é selecionado para ser executado.
